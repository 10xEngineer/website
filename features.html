Discover Labs!


Develop, test and validate your application and ideas faster with Labs - your personal sandbox available right when you need it. Whatever you want setup development environment for a new project, reproduce and fix known issue or try new software package in isolated and secure manner, Labs are just one command away.

Behind every lab is a Lab Machine, our custom lightweight virtualized server packaged with extra features to help you in everyday work.

Instant and easy to setup

No more waiting for server setup. Starting a new Lab Machine is matter of seconds. Control the full lifecycle with our command-line interface (CLI) or integrate Labs into your workflow using REST API. Using Lab Machines

  • improve your Continous Integration (CI) processes with faster runtime of automated tests.
  • provide same environment throughout the development lifecycle.

Snapshot at any point in time

Return to any given state of the Lab Machine repeatedly with snaphots. Analyse the what changed in time and understand the behaviour of the applications and systems. Down to the individual files.

  • Capture the exact state of the machine at the moment the error has occured for future reference.
  • Focus on your work, rather than fixing mis-configurations. Analyse changes compared to the last known working state. Revert instantly, you don't even have to logout.

No infrastructure to manage

With Labs as a hosted service you can forget infrastructure. We manage all the nitty-gritty of day-to-day operations, enabling you to focus on cool experiments.
